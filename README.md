# merchants
An offline merchant building tool for Paizo's Pathfinder 2nd Edition.

# Installation
Download the zip file, extract it where you like.

# Running
Open index.html to run.
The Merchant Generator runs right in your browser.


# Features

No logins required. No complicated installations.
No pestering for updates (the update button lives in Settings menu waiting on you).
It runs right in your browser and is lightning fast.
Have a merchant the minute you need it.
The merchant generator has access to the entirety of Paizo's published equipment catalog. Over 5600 items.
You can create custom items whole-cloth, or modify existing items. Modified items are assigned a unique identifier
so the published item is *never* lost in the process.

Merchants can be customized with multiple variables:
  - Location size
  - Location wealth
  - Merchant's race
  - The type of merchant (general goods, blacksmith, alchemist, etc...)
  - How focused a merchant is on their type. A focused blacksmith will only have items they make.
        A generalist blacksmith will have related items, but not necessarily things they can make themselves. Ex: maybe a few healing potions.
  - Arcane Tilt: a slider that determines what percentage of a merchant's inventory should be magic items.
  - Items level and rarity.
  - A base price modifier to raise or lower the price of all items in a merchant's inventory.
  - The ability to import/export custom items and merchants from .json files.
